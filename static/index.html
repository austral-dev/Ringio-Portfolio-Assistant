Características Principales

🧠

Sistema RAG Integrado

Las consultas se enriquecen dinámicamente con contexto en tiempo real del mercado y su portfolio.

⚡

Datos en Tiempo Real

Integración directa con APIs financieras líderes para mantener las cotizaciones siempre actualizadas.

🔒

Seguridad JWT

Autenticación robusta basada en tokens JSON Web (JWT) para proteger el acceso a sus datos.

¿Cómo Funciona el Asistente?

01

Autenticación & Consulta

El usuario se autentica de forma segura y envía una pregunta sobre su portfolio de inversión.

➔
02

Recuperación de Datos

El sistema consulta las cotizaciones y noticias más recientes de los activos involucrados.

➔
03

Generación Enriquecida

El modelo generativo (Gemini) procesa la información para dar una recomendación fundamentada.

Endpoints de la API

GET /health Estado del servidor y conexión.
POST /api/v1/auth/register Registro de nuevo usuario.
POST /api/v1/auth/login Inicio de sesión (obtiene token JWT).
POST /api/v1/ai/query Consulta RAG al asistente inteligente.
